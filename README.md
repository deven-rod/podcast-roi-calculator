# Podcast ROI Calculator

Standalone static site (single `index.html`, no build step, no dependencies) — an interactive calculator showing prospects the revenue potential of podcast guesting.

Open it by double-clicking `index.html`, or serve the folder with any static host.

To set the real booking link, edit the `href="#"` on the `<a id="ctaLink">` element near the bottom of `index.html` (it auto-adds `target="_blank"` once the href isn't `#`).
